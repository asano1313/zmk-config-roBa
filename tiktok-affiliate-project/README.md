まず「バズりやすい動画の調査」から着手し、調査結果をこのリポジトリに蓄積していきます。

## 🎯 このプロジェクトのゴール

1. **調査（今ここ）** — バズる動画の構造・トレンド・勝ちニッチを把握する
2. **設計** — 自分が狙うニッチ／キャラ／導線（リンク設置方法）を決める
3. **量産** — 検証済みテンプレで動画を週3〜5本投稿し、データで改善する
4. **収益化** — TikTok Shop アフィリエイト or 外部ASPリンクで収益を発生させる

## 📌 調査の要点（3行サマリー）

- **最初の3秒（10〜14語）のフック**が全て。ここで止められなければ動画は死ぬ。
- 勝ち筋は **ソフトセル（自然な紹介）× トレンド便乗 × マイクロインフルエンサー的な距離感**。
- 構成の黄金型は **フック → 価値（実演）→ ループ/CTA**。1本で完結する再現可能な型を回す。

詳細は各 `research/` ファイルを参照してください。
