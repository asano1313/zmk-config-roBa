# 03. 実践テンプレート（台本・運用）

調査結果（01・02）を「すぐ動ける形」に落とし込んだテンプレート集。

---

## 1. 台本テンプレート（コピペ用）

### A. 王道：問題解決型（アフィリエイト鉄板）

```
【0-3秒｜フック】
「○○で悩んでる人、これ知らないと損します」
（※冒頭からトレンドサウンドを鳴らす／結果を先に見せる）

【3-10秒｜問題提起】
「私もずっと△△で困ってた。□□を試しても全然ダメで…」

【10-35秒｜実演デモ】
「で、見つけたのがコレ。（商品を手に取り使ってみせる）
 ここがすごくて〜（ベネフィットを1〜2個に絞って実演）」

【35-45秒｜結果（ビフォーアフター）】
「結果こうなりました。（変化を視覚で見せる）」

【ラスト3-5秒｜CTA＋ループ】
「やり方はプロフのリンクから。保存しておくと後で困らないよ」
```

### B. 逆張り型（リーチ重視）

```
【フック】「その○○のやり方、9割の人が間違ってます」
【展開】「正しくはこう。理由は〜」（断言→根拠→実演）
【CTA】「他の間違いはコメントで聞いて／プロフに詳細」
```

### C. リスト型（保存されやすい）

```
【フック】「買ってよかった○○、勝手にベスト3」
【本編】3位→2位→1位とテンポよく。各商品10秒以内
【CTA】「全部プロフのリンクにまとめてます」
```

## 2. 週間投稿カレンダー（雛形）

| 曜日 | 投稿時間 | 形式 | ニッチ内テーマ |
| --- | --- | --- | --- |
| 月 | 18-21時 | 問題解決型(A) | 定番商品の実演 |
| 火 | — | （企画・撮影日） | — |
| 水 | 18-21時 | 逆張り型(B) | よくある間違い |
| 木 | — | （編集日） | — |
| 金 | 18-21時 | リスト型(C) | おすすめまとめ |
| 土 | 15-20時 | トレンド便乗 | 流行音源＋商品 |
| 日 | 15-20時 | 舞台裏/日常 | 親近感づくり |

> まずは **週3〜5本** から。10本目以降に伸び始めるのが普通なので、最低30本は続ける前提で。

---

## 3. 1本ごとの制作フロー

1. **ネタ選定** … トレンド音源・他人のバズ動画を毎日10本リサーチ（後述の調査シート）
2. **フック決定** … 01の型から1つ選び、10〜14語で言い切る形に
3. **台本** … 上のテンプレA/B/Cに当てはめる
4. **撮影** … 最初の2秒に「結果・完成・変化」を必ず入れる
5. **編集** … ムダなカット削除、テンポ重視、字幕、冒頭から音源
6. **投稿** … ゴールデンタイム＋関連ハッシュタグ＋CTA
7. **計測** … 完視聴率・保存数・クリック数を記録 → 次に反映

## 5. KPI（最初の指標）

| 指標 | 初期目標 | 意味 |
| --- | --- | --- |
| 平均完視聴率 | 50%以上 | フックと構成の良し悪し |
| 保存率 | 1%以上 | 「役立つ」と思われたか |
| プロフ遷移率 | 動画視聴の数% | CTAが効いているか |
| リンククリック | 計測必須 | 収益の手前の数字 |
| 投稿本数 | 月12〜20本 | 量がアルゴリズム学習を進める |

---

## 6. 注意点（やりがちな失敗）

- ❌ 無音スタート → 冒頭からトレンド音源を
- ❌ 自己紹介や前置きが長い → 0秒から本題
- ❌ ベネフィットを盛り込みすぎ → 1動画1メッセージ
- ❌ ジャンルが毎回バラバラ → アルゴリズムが配信先を学習できない
- ❌ 数本で諦める → 10本目以降が本番
