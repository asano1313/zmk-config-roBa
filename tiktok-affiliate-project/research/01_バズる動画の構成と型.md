# 01. バズるTikTok動画の構成と型（2026年版）

バズる動画は「センス」ではなく「構造」で作れる。ここでは2026年時点で再現性が確認されている
構成・フックの型・心理トリガー・音声戦略をまとめる。

---

## 1. 動画の黄金構成フレーム

バズる動画はほぼ例外なく次の4ブロックで出来ている。

```
① フック（0〜3秒）       … スクロールを止める。10〜14語で言い切る。
② 価値（3〜45秒）        … 実演・解説・比較で「見続ける理由」を渡す。
③ ループ / クリフハンガー … 最後をオチor余韻にして「もう一回見る／コメントしたくなる」状態に。
④ CTA（最後の3〜5秒）    … 保存・シェア・プロフのリンクへ誘導。
```

アフィリエイト動画向けの定番スクリプト型（コンバージョン重視）:

```
フック → 問題提起 → 実演（デモ）→ 結果（ビフォーアフター）→ CTA
```

> **完視聴率（最後まで見られた割合）が最重要KPI。** TikTokのアルゴリズムは
> 「3秒リテンション」と「完視聴率」を最優先で評価する。短くテンポよく、ムダなカットを削る。

---

## 2. 最初の3秒で勝負が決まる ―「3秒ルール」

- フックは **最初の3秒以内・10〜14語** に収める。
- ここで止められなければ、その動画は事実上「死ぬ」。
- **34,635本のクリップ分析** で最も成績が良かったフックは
  **「Product / Outcome Showcase」＝完成品・変化・結果を最初の2秒で見せる** タイプ。
  （例: いきなり完成した部屋／ビフォーアフター／驚きの結果から始める）

---

## 3. 2026年に効くフックの型（コピペで使える）

スコアが安定して高い（70点超）フックは主に4パターン。

| 型 | 狙い | 例（日本語に翻案） |
| --- | --- | --- |
| **アイデンティティ・コール** | 「自分のことだ」と思わせる | 「TikTokで物を売ってる人、これだけは知っといて」 |
| **反論・逆張り（Contrarian）** | 常識を否定して注意を奪う | 「60秒動画、もう投稿やめて」「その掃除のやり方、間違ってます」 |
| **オープンループ** | 続きが気になる未完の提示 | 「この1,400円のリップ、デパコス◯◯を置き換えます」 |
| **告白（Confession）** | 失敗談で共感と興味 | 「4,000円ドブに捨ててやっと気づいた話」 |

さらに王道の3型:

1. **Contrarian Claim（逆張り主張）**
2. **Mistake Warning（失敗・損の警告）**
3. **List Tease（リストの予告）**「知らないと損する3つの○○」

日本語のタイトル/サムネで効くフレーズ:
- 「知らない方がよかった○○」
- 「あなたの○○、間違っています」
- 「見ないと損する○○」

---

## 4. なぜ効くのか ― 4つの心理トリガー

すべての強いフックは、最低1つ（強いものは2つ以上を重ねている）のトリガーを踏んでいる。

1. **好奇心（Curiosity）** … 続きが知りたくなる
2. **パターン中断（Pattern Interrupt）** … 予想を裏切って手を止めさせる
3. **自分ごと化（Self-relevance）** … 「これ自分の話だ」
4. **感情の喚起（Emotional Arousal）** … 驚き・共感・怒りなど感情が動く

> 「意外性」「ストーリー性」「感情が動く」動画は "また見たい" を生み、再生とシェアに直結する。

---

## 5. 音声（サウンド）戦略

- 2026年のアルゴリズムは **オーディオ・リテンション（音声の続聴）も重視**。
- **最初の3秒で "聞き覚えのある音" を流すと、5秒以降まで残る確率が約40%上がる** というデータ。
- → **トレンドサウンドを冒頭から使う**のが鉄板。無音スタートは不利。

---

## 6. 投稿タイミング（日本）

初動の伸びがアルゴリズム評価を左右するため、ユーザーが見ている時間に投稿する。

- **平日: 18〜21時**
- **土日: 15〜20時**

---

## 7. すぐ使えるチェックリスト

- [ ] フックは3秒以内・10〜14語で言い切っているか
- [ ] 最初の2秒に「結果・完成品・変化」を見せているか
- [ ] 冒頭からトレンドサウンドが鳴っているか
- [ ] 心理トリガーを2つ以上重ねているか
- [ ] 価値パート（実演）は短くテンポ良いか（ムダなカットを削ったか）
- [ ] 最後はループ／クリフハンガーで終わっているか
- [ ] CTA（保存・シェア・リンク誘導）を入れたか
- [ ] 投稿時間はゴールデンタイムか
